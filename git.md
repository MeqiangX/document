### git常用操作和命令

1. 几个概念

   > Workspace：工作区
   >
   > Index/stage：本地暂存区
   >
   > repository：本地仓库
   >
   > remote：远程仓库

   ![img](https://cdn.jsdelivr.net/gh/MeqiangX/cloud-img@master/bg2015120901.png)

2. 新建仓库

   ```shell
   # 初始化一个本地git仓库 repository
   $ git init [项目(仓库)名称]可选参数
   # 下载一个远程仓库到本地
   $ git clone 远程代码仓库url
   ```
